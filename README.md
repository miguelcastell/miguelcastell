## 📌 O que tenho desenvolvido

- **Análise e Detecção de Fraudes Financeiras**: Projeto de Engenharia de Dados focado na criação de um pipeline ETL para análise e detecção de fraudes em transações financeiras.
- **Sistemas de Recomendação**: Desenvolvimento de sistemas de recomendação de filmes utilizando Python e Machine Learning para sugestões personalizadas.
- **Automação de Processos**: Criação de scripts em PowerShell para automatizar a instalação de programas essenciais em novos PCs e projetos em Python para automatizar a criação de documentos Word e PDF.
- **Gestão de Dados**: Implementação de sistemas de gestão de pedidos em PostgreSQL com procedures e triggers para automação de fluxos comerciais.
